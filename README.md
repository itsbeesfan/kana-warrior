# kana kenshin
### a game to learn japanese kana!!

-----
### what is this?

this is a quiz game that will help you study learn japanese kana! you play as a little fella, who, as the
levels progress, becomes more experienced until it becomes a sensei! train in the dojo, the countryside, the big city... you'll see!

**_"cry in the dojo, laugh in the arena"_ - my grade 7 maths teacher**

### okay, but what does it have?

- start screen with settings so you can adjust volume (and more coming soon!)
- sfx and music (with music by the wonderful newjeans, hybe please don't sue me :<)
- 5 different levels with increasing difficulty
  - level 1: dojo - nothing, just a _hachimaki_ (first 5 hiragana)
  - level 2: forest - _kasa_ hat (first 15 hiragana)
  - level 3: seaside - full ninja fit (first 25 hiragana)
  - level 4: city - samurai (first 35 hiragana)
  - level 5: city - samurai (45 hiragana!!!)
- fun "level completed" and "game over" screens!
- custom art!!! (yes, ALL made by me hehe)

### why? and also, what did i learn?
i really want to learn japanese. one of the first steps is to learn hiragana and katakana, and i felt like
making a quiz game to help me remember them would be perfect. ive only done hiragana up to now, but we'll see how 
it comes along in the future!!!

#### what did i learn?
- quiz game: so yes, this is mostly UI elements, and it was a change from tutorial-hell platformers and such.
- art: each project i create makes me a little better at art. and you can actually see the progression! from the very first
background i made (level 1) all the way to the last (city background, which i think looks SICKK)
- export issues: so exporting this was such a pain since the hiragana wouldn't show up in the itch.io web export. i ended 
up learning more about godot while trying to fix the issue.
- global data: a lot of this project is global data, so because of this, i learnt ways i could use global data.
